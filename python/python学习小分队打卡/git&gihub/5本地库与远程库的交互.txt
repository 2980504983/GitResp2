创建远程库:
    github -> 加号 -> new repository

远程库地址:
    进入github首页，左边就可以看到你创建的库，点进去，然后上面就有地址了
    远程库地址比较长，复制比较麻烦，git可以将地址保存，并且可以取一个别名，可以通过别名访问

在本地创建远程库地址别名:
    查看别名: git remote -v
    取别名: git remote add 你要取得名字 远程库的地址

将本地库内容提交到远程库:
    git push 别名 当前所在的分支名(没有另外创建分支的话一般是主分支master)

克隆操作:
    1 进到你要存储的文件夹
    2 打开git终端
    3 输入 git clone 要克隆的远程库地址(用github很慢，可以用码云)

克隆操作完成的事:
    1 把存储文件夹自动初始化为本地库
    2 将远程库内容克隆到本地
    3 创建别名(这个别名不是你创建的，是作者，你可以通过git remote -v查看别名)

邀请加入团队:
    远程库并不是随便就能push的，一般要输入加入团队者的账号和密码。(如果你的github账号没有加入团队就会push失败)
    而当你输入了一次密码后，git会自动保存，在同一台机器上即使没有得到邀请也可以不用在输入密码了
    删除git的自动保存 -> win键 -> 在里面点搜索图标 -> 管理你的凭据
    邀请成员 github仓库 -> settings -> manage access -> invite ... -> 输入被邀请者的名字 ->
    发给它邀请链接 -> 他讲链接输入在地址栏就行了
