【初始化本地仓库】
    {1} 创建一个文件夹:

    {2} 打开git终端:
        进去后先对字体和编码进行设置：
            (右键-options-text中character set选择UTF-8)

    {3} 设置签名:
        设置用户名:
            git config --global user.name "要填入的用户名"
        设置邮箱:
            git config --global user.email "要填入的邮箱"

    {4} 本地仓库的初始化操作:
        1 cd 到创建的文件夹中(也就是你要定义的仓库中)
        2 git init (创建了一个隐藏的.git文件)
        注：git目录下的文件不能随意修改
